# 🎨 OpenGraph Image Design Guide — JIBB Website

**Purpose:** Visual specifications for creating professional social media preview cards  
**Dimensions:** 1200×630px (1.91:1 ratio) — Universal for all platforms  
**Format:** JPG (optimized <1MB)  
**Total Images Needed:** 7 images

---

## 🎯 Design System Overview

### Brand Colors to Use
- **Primary Orange:** #FF6B35 (JIBB Orange)
- **Secondary Indigo:** #2E4B7F (JIBB Indigo)
- **Accent Sakura:** #E85D75 (JIBB Sakura/Pink)
- **Background Dark:** #1A1A1A or #0F0F0F
- **Text White:** #FFFFFF (for contrast)
- **Accent Light:** #F5F5F5

### Typography
- **Headlines:** Bold sans-serif (60-72px) — e.g., Montserrat Bold
- **Taglines:** Medium sans-serif (28-36px) — e.g., Plus Jakarta Sans Medium
- **Small Text:** Regular sans-serif (16-20px) — e.g., Inter Regular
- **All text in white** for maximum contrast on dark backgrounds

### Logo/Branding
- **JIBB Logo:** Place in top-left or top-right corner (150×150px max)
- **Logo Scale:** Should be recognizable at thumbnail size
- **Background:** Ensure logo has sufficient contrast

### Layout Safe Area
- **Full Image:** 1200×630px
- **Safe Zone (important content):** Center 1200×600px
- **Margin:** 15px padding from all edges
- **Avoid placing critical text within 100px of edges**

## 📐 Technical Requirements

### All Images
- **Dimensions:** 1200×630px (EXACT)
- **Aspect Ratio:** 1.91:1 (do NOT scale)
- **Format:** JPG (not PNG)
- **Compression:** Optimize to <1MB (target 800KB-950KB)
- **Color Space:** sRGB (for web)
- **DPI:** 72dpi (for web)

### Text Specifications
- **Headlines:** 60-72px bold
- **Subheadings:** 28-36px medium
- **Body Text:** 16-24px regular
- **All white text** for dark backgrounds
- **All dark text** for light backgrounds
- **Line height:** 1.2-1.4

### Image Requirements
- **JIBB Logo Size:** 140-160px width
- **Icons/Graphics:** 60-80px for medium elements
- **Safe Margin:** 15px from all edges
- **No critical text:** Within 100px of edges

## 🎯 Best Practices

### What Works Well
✅ **Bold headlines** that stand out at thumbnail size  
✅ **High contrast** between text and background  
✅ **Consistent branding** (logo on every image)  
✅ **Emotional connection** (trust, growth, innovation)  
✅ **Clear hierarchy** (main message first)  
✅ **Visual balance** (not too cluttered)  
✅ **Professional colors** from JIBB palette  

### What to Avoid
❌ **Too much text** (max 3-4 lines)  
❌ **Small logos** (must be recognizable)  
❌ **Poor contrast** (white on white, black on black)  
❌ **Overly complex** graphics  
❌ **Non-brand colors** or inconsistent style  
❌ **Generic stock photos** (use custom graphics)  
❌ **Centered everything** (use visual hierarchy)

## 🔧 Tools Recommended

### Design Tools
1. **Figma** (recommended - free tier available)
   - Professional design tool
   - Easy collaboration
   - Template creation
   - Size verification

2. **Canva Pro** (user-friendly)
   - Template-based design
   - Pre-built layouts
   - Good for quick creation
   - Limited customization

3. **Adobe Express** (cloud-based)
   - Easy to use
   - Brand kit support
   - Professional templates

4. **Photoshop/GIMP** (advanced)
   - Full control
   - Professional results
   - Steeper learning curve

### Optimization Tools
- **TinyJPG/TinyPNG** - Compress images
- **ImageOptim** - Batch optimization
- **Squoosh** - Google's image optimization
- **FileSize** - Check final size
